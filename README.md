## ⚡ Quick Start

</div>

Get up and running in your local environment in under 5 minutes:

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/Suman18-bit/Artificial-Intelligence.git
cd Artificial-Intelligence
```

### 2️⃣ Set Up Virtual Environment

```bash
# Create virtual environment (using uv)
uv venv

# Activate
.venv\Scripts\activate      # Windows
source .venv/bin/activate   # Linux / macOS
```

### 3️⃣ Install Dependencies

```bash
uv pip install -r Requirements.txt
```

> 💡 **Note:** Set your `MISTRAL_API_KEY` (and any other required credentials) as environment variables before running LangChain/RAG scripts.

### 4️⃣ Explore the Modules

```bash
# Explore LangChain experiments
cd Langchain

# Run a RAG pipeline
cd RAG

# Try out Runnables & Chains
cd "Runnables(Chains)"
```
